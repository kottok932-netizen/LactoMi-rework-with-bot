# Продолжение диалога в блоке ответа

Что добавлено:

- После первой расшифровки анализа в правом блоке активируется поле «Продолжить чат по этому анализу».
- Пользователь может задавать уточняющие вопросы без повторной загрузки файла.
- Чат опирается на контекст первой расшифровки и короткую историю последних сообщений.
- Новые сообщения не требуют повторного прохождения Turnstile: после первой успешной проверки сервер выдаёт временную подписанную сессию чата.
- Сессия чата действует около 30 минут. Если она истекла, нужно загрузить анализ заново.

Изменённые файлы:

- `index.html`
- `assets/site.js`
- `assets/main.css`
- `functions/api/analyze.js`

Переменные окружения:

- Ничего нового обязательно добавлять не нужно.
- По умолчанию для подписи временной чат-сессии используется `TURNSTILE_SECRET`.
- Если хочется отдельный секрет, можно добавить в Cloudflare Pages переменную `CHAT_SESSION_SECRET`.

Как проверить:

1. Загрузить PDF или фото анализа.
2. Пройти Turnstile.
3. Дождаться первой расшифровки.
4. Внизу правого блока должно активироваться поле чата.
5. Написать вопрос, например: «Объясни Lactobacillus простыми словами».
6. Проверить, что ответ появляется в том же чате без повторной загрузки файла.

Важно:

- Чат не ставит диагноз и не должен назначать лечение.
- Ответы строятся только на первой расшифровке и истории текущего диалога.
- Если Cloudflare Workers AI недоступен или не подключён, первая расшифровка может работать по правилам, но продолжение чата будет недоступно.
