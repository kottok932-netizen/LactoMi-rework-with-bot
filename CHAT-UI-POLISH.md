# Compact chat UI update

Изменения:

- убрана лишняя внешняя рамка у сообщений чата;
- follow-up ответы бота теперь выводятся компактнее;
- пользовательские сообщения занимают меньше места;
- поле продолжения чата оформлено как кастомный composer;
- Enter отправляет сообщение, Shift+Enter переносит строку;
- textarea автоматически подстраивает высоту под текст.

Основные файлы:

- `index.html`
- `assets/site.js`
- `assets/main.css`
