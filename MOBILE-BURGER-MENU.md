# Mobile burger menu

Добавлено компактное мобильное меню для шапки сайта.

Что изменено:

- На экранах до 760px навигация и кнопка темы скрываются в бургер-меню.
- В шапке остаются только логотип LactoMi и компактная кнопка меню.
- При нажатии меню раскрывается выпадающей панелью.
- При выборе пункта меню панель закрывается автоматически.
- Работает на index.html, guide.html и privacy.html.

Файлы:

- index.html
- guide.html
- privacy.html
- assets/main.css
- assets/mobile-menu.js
